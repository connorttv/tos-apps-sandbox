# Tele-Menu Application

A dynamic digital menu board application with customizable settings and smooth animations.

## Features & Settings

All configurations are managed via the **Settings** view.

###  General
- **Menu Title**: Set a custom header label (e.g., "BREAKFAST", "LUNCH") displayed on the sidebar.

###  Display
- **UI Scale**: Adjust the overall size of the interface (1x to 3x) to fit different distinct screens.
- **Rounded Corners**: Toggle rounded corners for the main window frame.
- **Dark Mode**: Switch between Dark (high contrast) and Light themes.

###  Menu Items Management
Manage your food items with an intuitive **accordion interface**:
- **Add / Remove Items**: dynamic list management.
- **Reorder**: Use ↑ / ↓ buttons to change item sequence.
- **Item Details**:
  - **Name & Price**: Standard menu details.
  - **Description**: Optional text for item ingredients or details.
  - **Images**:
    - Choose from built-in high-quality assets.
    - **Custom URL**: Paste a link to your own image (overrides built-in selection).
    - **Fallback**: Automatically shows a placeholder if no image is provided.
- **Visibility**:
  - **Featured**: Highlights the item with a special badge and border.
  - **Hide**: Temporarily remove an item from the grid without deleting it.

###  Visual Customization
- **Accent Color**: Customize the primary color used for prices and highlights.
- **Featured Styling**: Set a custom border color and badge text (e.g., "NEW", "HOT") for featured items.
- **Logo**: Add a custom brand logo URL to the sidebar.
- **Promo Message**: Display a custom promotional text area.

###  Section Rotation
- **Auto-Rotation**: Automatically cycle through pages of menu items if they exceed the screen capacity.
- **Smooth Transitions**: Items fade in smoothly when the section changes.
- **Duration**: Adjustable rotation interval (5s - 60s).
